
Code_For_Report_v3 is the only file that is needed to run. 

To run the file following is required:

1. input folder with .in files. In the following library input 
   folder is named as 1_input_practiceInsideFolder

2. Gurobi needs to be installed:

	Gurobi offers a free academic license. 
	A quick Google search should provide all the instructions for getting it up 
	and running. Basically make an account, download, and then set the key.
	
3. Other functionality includes:
	
	pip install numpy
	pip install networkx
	pip install matplotlib
	pip install ortools
	pip install plotly
